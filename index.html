Summarize With AI

A lightweight, embeddable widget for summarizing pages with AI

Light Theme (Default)

The default appearance with all AI services. Click any button to summarize this page.

<div id="summarize-widget"></div> <script> SummarizeWidget.init({ target: '#summarize-widget', theme: 'light' }); </script>

Dark Theme

Perfect for dark mode websites. Uses inverted colors for better contrast.

<summarize-with-ai theme="dark"></summarize-with-ai>

Minimal Theme

A subtle, borderless design that blends into any page without the header.

<summarize-with-ai theme="minimal"></summarize-with-ai>

Custom Services

Show only specific AI services. Great for keeping things focused.

SummarizeWidget.init({ target: '#widget', services: ['chatgpt', 'claude', 'perplexity'] });

Content Mode with Selection

Try selecting some text below, then click a button. The selection will be used for summarization!

Sample Article Content

Artificial intelligence has transformed how we interact with technology. From voice assistants to recommendation systems, AI is now embedded in our daily lives. Machine learning algorithms can analyze vast amounts of data to find patterns that humans might miss. Natural language processing enables computers to understand and generate human language with increasing sophistication. As these technologies continue to evolve, they promise to revolutionize fields from healthcare to education, making previously impossible tasks routine.

SummarizeWidget.init({ target: '#widget', mode: 'content', preferSelection: true, maxChars: 4000 });

Custom Prompt Prefix

Add context to your summarization requests with a custom prompt prefix.

SummarizeWidget.init({ target: '#widget', promptPrefix: 'As a tech enthusiast, summarize the key insights from:' });

Web Component

Use as a custom HTML element for maximum simplicity.

<summarize-with-ai theme="light" services="chatgpt,claude,gemini" mode="url" ></summarize-with-ai>
