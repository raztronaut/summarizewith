# SummarizeWith Mockup Tool

A standalone script to help you visually mockup and test the `summarize-with-ai` widget on any website.

## How to Use
1. Open [`universal_mockup.js`](./universal_mockup.js) and copy the **entire content**.
2. Go to the website where you want to test the widget.
3. Open the **Developer Console** (F12 or Right Click > Inspect > Console).
4. Paste the code and hit **Enter**.
5. Use the control panel to configure and place widgets.

## Features
- **Visual Interface**: Premium Apple-style UI.
- **Theme & Alignment**: Toggle themes (Light/Dark/Minimal) and alignment (Left/Center/Right).
